自定义组件的使用者似乎无法直接在组件上加行内样式，例如：
```js
function Child() {
  return <div>I'm the child</div>
}
function Parent() {
  return <Child style={{ display: 'none' }}/>
}
```
这种情况通过styled-component可以解决:
```js
const Hidden = styled.div`
  display: none;
`
function Child() {
  return <div>I'm the child</div>
}
function Parent() {
  return (
    <Hidden>
      <Child />
    </Hidden>
  )
}
```
